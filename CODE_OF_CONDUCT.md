# Code of Conduct

## Our Pledge

This is a private, personal project. Anyone granted access to this repository agrees to treat all content — including personal messages, photos, and private letters — with full respect and confidentiality.

We pledge to make participation in this project a respectful and harassment-free experience for everyone.

## Our Standards

**Acceptable behaviour:**
- Keeping all personal content in this repository strictly private
- Providing constructive feedback via the pull request process
- Respecting the personal nature of the content

**Unacceptable behaviour:**
- Sharing, distributing, or republishing any personal content without explicit consent
- Harassment, insults, or derogatory comments toward any person mentioned in the project
- Accessing or modifying the admin panel without authorisation

## Enforcement

Violations may result in immediate removal of repository access.
